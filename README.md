Telzir
***
Página web para os clientes da empresa calcularem o valor de suas ligações. Aplicação desenvolvida em Python utilizando o framework Flask.


Inicio:

Criar o diretório.

# Instalar todas as dependências.
$ pip install -r requirements.txt
